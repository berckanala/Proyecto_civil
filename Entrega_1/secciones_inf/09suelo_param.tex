\subsection{Clasificación Suelo y descripción de características generales}

Segun la norma NCh433, el tipo de suelo para la zona de antofagasta es de tipo A, lo cual se corresponde con roca o suelo cementado. En terminos sismicos, este suelo presenta una buena respuesta ante ondas sismicas, ya que al no tratarse de un suelo blando, no amplifica las ondas sismicas, lo que se traduce en una menor aceleración del suelo durante un evento sísmico. \subsection{Constante de balasto}

\textbf{NESECITAMOS DATOS DE LAS FUNDACIONES PARA CALCULAR LA CONSTANTE DE BALASTO}

La constante de balasto ($k$) es un parámetro que se utiliza para describir la relación entre la presión aplicada en la base de una cimentación y la deformación resultante del suelo. Este valor depende del tipo de fundacion, asi como las dimenciones de esta, ademas de las propiedades del suelo. En la etapa actual del proyecto, no se han definido las caractiristicas dimencionales de las fundaciones, pero si se puede establecer que los pilares usaran zapatas aisladas y los muros usaran zapatas corridas. Para el caso de las zapatas aisladas, la constante de balasto se puede estimar utilizando la siguiente formula:

\begin{equation}
k = \frac{E}{B}
\end{equation}

Donde $E$ es el módulo de elasticidad del suelo y $B$ es el ancho de la zapata. Para el caso de las zapatas corridas, la constante de balasto se puede estimar utilizando la siguiente formula:

\begin{equation}
k = \frac{E}{L}
\end{equation}

Donde $L$ es la longitud de la zapata corrida.
