\section{MATERIALES}

Para la construcción del edificio se utilizará hormigón armado en los elementos estructurales principales, tales como muros, vigas y losas. La resistencia del hormigón se ha definido según la demanda estructural de los niveles:

\begin{itemize}
    \item \textbf{Subterráneo y niveles inferiores:} Se utilizará un hormigón de resistencia $f'_c = 35$ MPa.
    \item \textbf{Pisos superiores:} Se utilizará un hormigón de resistencia $f'_c = 25$ MPa.
\end{itemize}

Para la tabiquería de divisiones interiores, se empleará un sistema de perfiles de acero galvanizado (Metalcom) revestidos con planchas de yeso-cartón (Volcanita).

Respecto al acero de refuerzo, se especifica el uso de acero grado \textbf{A630-420H}. Esto implica que el material presenta las siguientes propiedades mecánicas:
\begin{itemize}
    \item \textbf{Límite de fluencia ($f_y$):} $420$ MPa.
    \item \textbf{Resistencia máxima a la tracción ($f_u$):} $630$ MPa.
\end{itemize}
