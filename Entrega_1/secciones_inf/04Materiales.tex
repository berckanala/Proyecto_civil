\section{MATERIALES}
Para la contrucción del edificio se utilizará hormigón armado para los elementos estructurales como muros, vigas y losas. Para el subterráneo y los primeros pisos del edificio se utilizará un hormigón de 35 Megapascales de resistencia, mientras que para los pisos superiores se utilizará un hormigón de 25 Megapascales de resistencia. Para las paredes exteriores se utilizará ladrillo cerámico hueco, mientras que para las paredes interiores se utilizará ladrillo cerámico macizo. Para la tabiquería se ocupará volcanita con metalcom.
